Installing Claude Code

In this lesson, you'll install Claude Code and all the tools you need to complete this course. Don't worry - we'll walk through everything step by step!

What You'll Install

  • Claude Code: The AI-powered assistant
  • Node.js: Required for Claude Code to run
  • A code editor: VS Code (optional but recommended)

System Requirements

Before we begin, make sure your computer meets these requirements:

  • Operating System: macOS, Windows 10+, or Linux
  • RAM: At least 4GB (8GB+ recommended)
  • Disk Space: 500MB free space
  • Internet: Stable internet connection

Step-by-Step Installation

1Create an Anthropic Account

Visit console.anthropic.com and create a free account. You'll need this to use Claude Code.

Once registered, navigate to your account settings to get your API key.

2Install Node.js

Claude Code requires Node.js. Download and install it from nodejs.org.

Choose the LTS (Long Term Support) version for your operating system.

To verify installation, open your terminal/command prompt and run:

node --version
npm --version

You should see version numbers displayed.

3Install Claude Code

Open your terminal/command prompt and run:

npm install -g @anthropic-ai/claude-code

This will install Claude Code globally on your system.

Mac/Linux users: If you get a permission error, try:

sudo npm install -g @anthropic-ai/claude-code

4Configure Your API Key

Claude Code needs your Anthropic API key to work. Run:

claude-code config set-key

When prompted, paste your API key from step 1.

Security Note: Your API key is stored securely on your local machine and is never shared.

5Install VS Code (Optional)

While not required, Visual Studio Code makes it easier to view and edit marketing files.

Download it from code.visualstudio.com.

After installation, you can open any folder with:

code .

Verify Your Installation

Let's make sure everything is working correctly. In your terminal, run:

claude-code --version

You should see the Claude Code version number. If you do, congratulations! You're all set up.

Understanding Your Setup

Now that you've installed Claude Code, let's understand what you have:

  • Terminal/Command Line: Where you'll interact with Claude Code
  • Claude Code CLI: The command-line interface for Claude
  • API Connection: Links your local setup to Claude's AI
  • VS Code (optional): For viewing and editing files

Common Installation Issues

Issue: "command not found: claude-code"

Solution: Make sure you installed with the -g flag. Try closing and reopening your terminal, or restart your computer.

Issue: "Permission denied"

Solution: On Mac/Linux, use sudo before the install command. On Windows, run Command Prompt as Administrator.

Issue: "API key invalid"

Solution: Double-check your API key from the Anthropic console. Make sure you copied it completely.

Still Having Issues?

Visit the Claude Code documentation or check the troubleshooting section.
