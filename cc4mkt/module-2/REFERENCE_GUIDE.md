# Module 2: Advanced Applications - Quick Reference

## Campaign Brief Creation

### Campaign Brief Template Structure
1. **Campaign Overview**: Name, duration, budget, goals
2. **Audience & Positioning**: Target segments, personas, competitive positioning
3. **Channel Strategy**: Budget allocation, expected ROI by channel
4. **Creative Strategy**: Key messages, creative concepts
5. **Execution Timeline**: Week-by-week plan with milestones
6. **Measurement Framework**: KPIs, success metrics, reporting structure

### Quick Prompt
```
Create a campaign brief for [product/service]:
- Campaign name: [name]
- Duration: [timeframe]
- Budget: $[amount]
- Primary goal: [metric and target]
- Target audience: [personas]
- Channels: [list]

Include all sections: overview, audience, positioning, channel strategy,
creative concepts, timeline, and measurement framework.
```

## Content Strategy

### Content Pillar Framework
```
Create [number] content pillars for [product/topic]:

For each pillar:
- Core theme and positioning
- 10-12 supporting topics
- Format mix (blog, video, guides)
- SEO keyword mapping
- Success metrics
```

### Content Calendar Structure
```
Create [timeframe] content calendar:
- Channels: [blog, email, social, etc.]
- Frequency: [posts per channel per week]
- Include: publication dates, topics, formats, keywords, status
- Map to content pillars
```

### Content Atomization
```
Take this core asset: [describe asset]
Atomize into [number] formats:
1. [Format 1] ([length/specs])
2. [Format 2] ([length/specs])
...

Create distribution plan for each channel.
```

## SEO Optimization

### Keyword Research Framework
```
Research keywords for [topic/product]:

Top of funnel (awareness):
- [Keyword 1]: [volume/mo, difficulty]
- [Keyword 2]: [volume/mo, difficulty]

Middle of funnel (consideration):
- [Keyword 1]: [volume/mo, difficulty]
- [Keyword 2]: [volume/mo, difficulty]

Bottom of funnel (decision):
- [Keyword 1]: [volume/mo, difficulty]
- [Keyword 2]: [volume/mo, difficulty]

For each cluster, provide:
- Search volume and trend
- Difficulty score
- SERP analysis
- Content opportunities
```

### On-Page Optimization Checklist
```markdown
Optimize [page/post] for keyword: "[target keyword]"

- [ ] Title tag (50-60 chars, includes keyword)
- [ ] Meta description (150-160 chars, compelling)
- [ ] URL (clean, descriptive, keyword-rich)
- [ ] H1 (single, includes keyword)
- [ ] H2/H3 structure (logical, keywords)
- [ ] Keyword in first 100 words
- [ ] 2-4 internal links
- [ ] 1-3 external authoritative links
- [ ] Image alt text (descriptive)
- [ ] Content length (800+ words for blog)
- [ ] Readability score (Grade 8-10)
```

## Quick Tips

### Campaign Planning
- Start with clear, measurable goals
- Align on personas and positioning first
- Allocate budget based on historical ROI
- Build in measurement from the start
- Plan for iteration and optimization

### Content Creation
- Strategic > random (pillars and calendars)
- One core asset â†’ atomize into many formats
- SEO integration from the start
- Consistent publishing schedule
- Measure performance, optimize continuously

### Copywriting
- Benefits > features always
- Specific numbers and outcomes
- Use proven formulas (AIDA, PAS, BAB)
- Address objections proactively
- Strong, clear CTAs

### Analytics
- Focus on business outcomes, not vanity metrics
- Multi-touch attribution for accuracy
- Find the insight, not just the data
- Communicate clearly to executives
- Recommend actions, not just report numbers

### Competitive Intelligence
- Update regularly (quarterly minimum)
- Focus on strategy, not just features
- Win/loss analysis reveals truth
- Battle cards empower sales
- Monitor changes and respond quickly

### SEO
- Keyword research first
- On-page optimization checklist
- Content pillars for authority
- Internal linking structure
- Regular content refresh
