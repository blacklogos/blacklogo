# Module 2: Advanced Applications - Teaching Guide

## Module Overview

This is where everything comes together. Students apply core concepts to complete, real-world marketing workflows.

**Student State at Module 2**:
- Comfortable with Claude Code basics
- Ready for practical application
- Want to see real ROI
- Thinking: "How do I use this in my actual job?"

**Goal**: By end of module, students can execute complete marketing workflows independently

## Module Philosophy

**Complete > Fragmented**: Teach end-to-end workflows
**Realistic > Simplified**: Use real complexity and constraints
**Strategic > Tactical**: Explain the "why" behind the "what"
**Applicable > Theoretical**: Everything should be usable Monday morning

- Go deeper on interested topics
- Provide advanced variations
- Share additional resources
- Challenge with complex scenarios

### If Student is Industry-Specific:
- Adapt all examples to their industry
- Use their competitors in examples
- Address industry-specific challenges
- Connect with industry best practices
