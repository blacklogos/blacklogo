## Part 2: Multi-Touch Attribution

### Understanding the Customer Journey
```
Analyze attribution data for 100 recent trial signups:

Journey example:
User 1: LinkedIn ad ‚Üí Blog post ‚Üí Email ‚Üí Trial signup
User 2: Google search ‚Üí Landing page ‚Üí Trial signup
User 3: Blog (organic) ‚Üí Email ‚Üí Webinar ‚Üí Trial signup

Attribution models to compare:
- First touch: What channel started the journey?
- Last touch: What channel converted?
- Linear: Equal credit to all touchpoints
- Time decay: More credit to recent interactions
- Position-based: More credit to first and last (40-20-40)

Calculate:
- Channel value under each model
- True cost per acquisition
- Multi-touch conversion paths
- Average touches to conversion

Create attribution analysis report.
```
