# Quick Start Guide

Get up and running with Claude Code for Marketers in under 5 minutes.

## ðŸš€ 5-Minute Setup

### Step 1: Open the Course
Open `course-index.html` in your web browser to see the full course curriculum.

### Step 2: Install Claude Code
Follow the installation instructions in Module 0, Lesson 0.1.

Quick version:
```bash
# Install Node.js from nodejs.org first, then:
npm install -g @anthropic-ai/claude-code

# Configure your API key
claude-code config set-key
```

### Step 3: Create Your Workspace
```bash
# Create a folder for your projects
mkdir claude-marketing-projects
cd claude-marketing-projects

# Start Claude Code
claude-code
```

### Step 4: Try Your First Task
In Claude Code, type:
```
Create a campaign brief template for a product launch
```

That's it! You're now using AI to assist with marketing tasks.

## ðŸ’¡ Pro Tips

1. **Be Specific** - "Write an ad" vs "Write a Facebook ad for remote team managers, 125 characters, emphasizing time savings"

2. **Provide Context** - Always mention your brand, audience, and goals

3. **Iterate** - Ask Claude to refine outputs: "Make it more engaging" or "Add statistics"

4. **Use Brand Guidelines** - Create a brand voice document and reference it in prompts

5. **Save Templates** - When Claude creates something great, save it as a template

## ðŸ†˜ Troubleshooting

**Claude Code won't start**
- Check Node.js is installed: `node --version`
- Check Claude Code is installed: `claude-code --version`
- Try reinstalling: `npm install -g @anthropic-ai/claude-code`

**Output isn't good**
- Be more specific in your prompt
- Provide more context about brand/audience
- Ask Claude to revise and improve
- Show examples of what you want

**Not sure what to ask**
- Start with: "I need help creating [thing] for [audience]"
- Ask Claude: "What information do you need from me to create this?"
- Use the example prompts in the lessons
